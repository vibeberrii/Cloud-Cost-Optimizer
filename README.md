# ☁️ Cloud Cost Optimizer

An AI-powered multi-cloud cost optimization platform that helps identify underutilized cloud resources, estimate potential savings, and generate intelligent FinOps recommendations.

## 🚀 Features

* Multi-Cloud Support (AWS, Azure, GCP)
* AI-Powered Cost Analysis using Gemini AI
* AWS EC2 Cost Calculator
* Resource Cost Tracking Dashboard
* Interactive Cost Visualization Charts
* Cost Optimization Recommendations
* Report Export Functionality
* Local Storage Persistence
* Add/Delete Cloud Resources

## 🛠️ Tech Stack

### Frontend

* React
* Recharts
* CSS

### Backend

* Node.js
* Express.js

### AI

* Google Gemini AI

## 📊 Project Capabilities

* Track cloud resources across multiple providers
* Calculate estimated cloud spending
* Identify underutilized resources
* Estimate potential cost savings
* Generate AI-driven optimization recommendations
* Export cloud cost reports
* Visualize cloud spending using charts

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/vibeberrii/Cloud-Cost-Optimizer.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Environment Variables

Create:

```env
backend/.env
```

Add:

```env
GEMINI_API_KEY=YOUR_API_KEY
```
